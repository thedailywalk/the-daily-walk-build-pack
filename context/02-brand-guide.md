# 02 — Brand Guide

## Colors (use as design tokens)
| Token | Hex | Use |
|---|---|---|
| Navy (primary) | `#1F3A5F` | Headings, masthead, primary buttons, dark sections |
| Navy deep | `#16304F` | Hover/active on navy |
| Gold (accent) | `#C9A24B` | Accent, highlights, featured button, pills |
| Gold deep | `#B8902E` | Gold text/links on white (better contrast) |
| Cream | `#FAF6EE` | Soft section backgrounds |
| Ink | `#22262B` | Body text |
| Grey | `#5A6270` | Secondary text |
| White | `#FFFFFF` | Backgrounds |
| Green | `#1f9d55` | "Yes"/checkmarks in comparisons |

## Typography
- **Headings:** warm serif — Instrument Serif, Playfair Display, or Georgia.
- **Body / UI:** clean sans — Inter (fallback: Helvetica, Arial, system-ui).
- Generous line-height; mobile-first sizing.

## Logo / masthead
- Wordmark: **THE DAILY WALK** (no logo file yet — render as styled text; a logo can be added later).
- Optional small eyebrow above headlines: "A daily devotional newsletter".

## Voice & tone
- Clear over clever. Short sentences. Plain English; gloss any church term.
- Encouraging, never guilt-driven. "Welcome back," never "you're behind."
- Welcoming to the unsure: write so a curious non-believer feels at home next to a lifelong Christian.
- Grounded in Scripture; one idea per piece.
- Warm, hopeful, human.

## Visual style
- Calm, hopeful imagery: sunrises, light, open paths, nature, mountains at dawn.
- Avoid busy/stocky "lifestyle" photos. Minimal and inspiring.
- Dark overlay on hero photos so white text stays readable.
