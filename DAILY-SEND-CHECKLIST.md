# 📋 The Daily Walk — Daily Send Checklist (~2 minutes)

The website writes & publishes the devotional on its own every morning.
This checklist is just the **email send** (manual until you upgrade Beehiiv).

## Every morning

1. **Open Devotional Prep** → go to `thedailywalknewsletter.com/admin` → sign in → today's issue is already written.
2. *(Optional)* Edit anything, or add a **"How Healing Works"** neuroscience blurb in that field.
3. Click **Copy email HTML** → it says **Copied ✓**.
4. In **Beehiiv** → **+ New → Blank draft post**.
5. **Hide the header bits:** click the **👁 eye icon** next to the **title** and next to **your name** (so only your masthead shows). *(Slashed eye = hidden ✓)*
6. Click the **"New post"** title and type a real **subject**, e.g. *"The Daily Walk · [date]"*.
7. Click in the body → type **`/`** → **HTML Snippet** → click inside the dark box → **paste** (`Cmd+V`).
8. Top-right → **Preview** → make sure it looks right (navy/cream/gold). 🎨
9. Click **Next** → step through **Audience → Email → Web → Review**.
10. **Send.** 🙌

## One-time, before you email real (non-you) subscribers
- Beehiiv → **Settings → Emails** → set a **physical mailing address** in the footer (US law; a PO box works).

## When you're ready to go fully hands-off
- Upgrade Beehiiv to the plan with **RSS-to-Send** → point it at
  `https://thedailywalknewsletter.com/devotional/rss.xml` → set a send time → enable.
- After that, **steps 3–10 disappear** — the email sends itself. You only edit if you want to.
